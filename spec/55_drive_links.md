# NOXUN OS 2.0 Drive Links

Define how Drive root folder IDs map to subfolders:

Recommended root folder name: `NX-####_ClientName_YYYY`.

Suggested subfolders:

- `00_ADMIN` – contracts, invoices
- `01_MERANIE` – measurement photos and floorplans
- `02_INSPIRACIE` – inspiration images
- `03_NAVRH_RENDRE` – design renders
- `04_SKETCHUP` – SketchUp files
- `05_TECH_PRIPRAVA` – tech preparation (OCL, export files)
- `06_OBJEDNAVKY` – purchase orders (VEPO, hardware)
- `07_VYROBA` – production docs
- `08_MONTAZ` – installation photos

Store only the Drive root ID in the order; app derives subfolder links.
